## DOMAIN KNOWLEGE
1. **КЛЮЧЕВЫЕ МЕТРИКИ:**
   — **Turnover rate**: Уволившиеся / Средняя численность × 100%. Бенчмарк: <10% годовых (healthy), 10–15% (watch), >15% (problem).
   — **Voluntary turnover**: По собственному желанию. Главный индикатор проблем.
   — **Time to hire**: Среднее дней от публикации вакансии до offer acceptance. Бенчмарк: 30–45 дней.
   — **Cost per hire**: (Рекрутинг + HR + время найма) / количество нанятых. Бенчмарк: $3000–$5000 (mid-level).
   — **Employee engagement**: Опросы (eNPS, pulse surveys). eNPS >30 — хорошо, <0 — кризис.
   — **Absence rate**: Пропущенные дни / плановые дни × 100%. >5% — проблема.
   — **Revenue per employee**: Выручка / количество сотрудников. Растёт = растёт производительность.

2. **ДАШБОРД HR:**
   — Headcount: текущее, по отделам, план vs факт.
   — Turnover: по месяцам, по отделам, по причинам.
   — Hiring funnel: вакансии → отклики → интервью → offer → onboard.
   — Engagement: eNPS trend, pulse survey results.
   — Diversity: по полу, возрасту, отделам (если данные есть).

3. **ПРИЧИНЫ ТЕКУЧКИ (по данным Gallup):**
   — #1: Отсутствие growth opportunities (32%).
   — #2: Менеджмент / руководитель (25%).
   — #3: Компенсация (18%).
   — #4: Культура / окружение (15%).
   — #5: Work-life balance (10%).
