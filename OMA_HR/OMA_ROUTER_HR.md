## ТАБЛИЦА МАРШРУТИЗАЦИИ

| Задача / Ситуация | Юнит |
|---|---|
| Написать вакансию, описание позиции, требования | **job-description** |
| Оценить резюме, отобрать кандидатов, screening | **resume-screening** |
| Подготовить вопросы для собеседования, scorecard, structured interview | **interview-kit** |
| План адаптации нового сотрудника, 30/60/90 дней | **onboarding-plan** |
| Должностная инструкция, SOP, регламент, процесс | **job-instructions** |
| Оценка эффективности, performance review, 1-on-1 | **performance-review** |
| Зарплатная вилка, бенефиты, компенсации | **compensation-benefits** |
| HR-метрики, аналитика, turnover, engagement | **hr-analytics** |
| Корпоративная культура, ценности, вовлечённость | **corporate-culture** |
| Увольнение, offboarding, exit interview, передача | **offboarding** |

## СЕРЫЕ ЗОНЫ

| Ситуация | Уточняющий вопрос | А → | Б → |
|----------|-------------------|-----|------|
| «Нанять сотрудника» | «Нужно написать вакансию — или провести собеседование?» | Вакансия → **job-description** | Собес → **interview-kit** |
| «Адаптация сотрудника» | «Это план на первые 90 дней — или должностная инструкция?» | План → **onboarding-plan** | Инструкция → **job-instructions** |
| «Оценка сотрудника» | «Performance review — или определить зарплату / продвижение?» | Оценка → **performance-review** | Компенсация → **compensation-benefits** |
