---

## DOMAIN KNOWLEDGE

1. **СТРУКТУРА:**
   — По отделам: Engineering, Marketing, Sales, HR, Finance.
   — По типу: How-to, Process, Reference, Policy.
   — По актуальности: Active, Archive, Deprecated.

2. **ИНСТРУМЕНТЫ:**
   — Notion: flexible, database-driven.
   — Confluence: enterprise, deep integrations.
   — GitBook: technical documentation.
   — Google Docs: simple, collaborative.

3. **ПРАВИЛА:**
   — Один источник правды: каждая тема — в одном месте.
   — Owner: каждый документ имеет ответственного.
   — Review cycle: раз в квартал — обзор актуальности.
   — Searchability: теги, заголовки, ключевые слова.
