## DOMAIN KNOWLEGE

1. **ЭТАПЫ ВОРОНКИ В CRM:**
   — Стандарт B2B: New Lead → Contacted → Qualified → Discovery → Demo → Proposal → Negotiation → Closed Won / Closed Lost.
   — SLA по этапам: New Lead → Contacted за 5 минут (auto-assignment). Qualified → Discovery за 3 дня. Proposal → Decision за 7 дней.
   — Автоматические переходы: email opened → Contacted. Demo booked → Discovery.
   — Closed Lost reasons: бюджет, timing, конкурент, не соответствует, нет ответа.

2. **КЛЮЧЕВЫЕ ПОЛЯ:**
   — Deal: название, сумма, этап, close date, source, owner, probability.
   — Contact: имя, email, телефон, должность, LinkedIn, роль в сделке (champion / blocker / economic buyer).
   — Company: название, отрасль, размер, сайт, ICP tier.
   — Activity: звонки, email, встречи, задачи — всё логируется автоматически.

3. **АВТОМАТИЗАЦИИ:**
   — Auto-assignment: новый лид → назначается SDR по round-robin или territory.
   — Task creation: Qualified → создать задачу «Провести discovery» с deadline 3 дня.
   — Email sequences: автоматические follow-up после discovery и demo.
   — Alerts: сделка на одном этапе > 2x среднего = уведомление менеджеру.
   — Deal stage automation: если все tasks completed → предложить переход на следующий этап.

4. **ДАШБОРДЫ:**
   — Pipeline overview: сумма по этапам, количество сделок, средний чек.
   — Activity dashboard: звонки, email, встречи по менеджерам.
   — Conversion funnel: конверсия по этапам за период.
   — Forecast: weighted pipeline (сумма × probability) на квартал.

5. **ВЫБОР CRM:**
   — **HubSpot**: Лучший для inbound + marketing integration. Free tier до 1000 контактов. $45–$1200/мес.
   — **Salesforce**: Максимальная кастомизация. Для enterprise. $25–$300/user/мес.
   — **Pipedrive**: Простой, визуальный пайплайн. Для SMB. $15–$99/user/мес.
   — **amoCRM**: Российский, сильный в мессенджерах. $15–$45/user/мес.

## АНТИ-ПАТТЕРНЫ — СПЕЦИФИЧЕСКИЕ

1. **СЛИШКОМ МНОГО ЭТАПОВ**: 15+ этапов = никто не обновляет. 6–8 — оптимально.
2. **НЕТ SLA**: Без сроков по этапам — сделки зависают навсегда.
3. **НЕТ АВТОМАТИЗАЦИИ**: Ручное создание задач и назначение лидов = ошибки и задержки.
