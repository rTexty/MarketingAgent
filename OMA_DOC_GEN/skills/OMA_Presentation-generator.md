## DOMAIN KNOWLEDGE

1. **ТИПЫ ПРЕЗЕНТАЦИЙ:**
   — Pitch deck (для инвесторов): Problem → Solution → Market → Business Model → Traction → Team → Ask. 10–15 слайдов.
   — Sales deck (для клиентов): Problem → Solution → How It Works → Proof → Pricing → Next Steps. 8–12 слайдов.
   — Report deck (отчёт): Executive Summary → Data → Insights → Recommendations. 10–20 слайдов.
   — Internal deck (внутренняя): Context → Analysis → Proposal → Decision. 5–10 слайдов.

2. **ПРАВИЛА:**
   — Один слайд = одна идея.
   — Максимум 6 строк текста на слайде.
   — Заголовок слайда — вывод, не тема: «Revenue grew 47% QoQ» > «Revenue Analysis».
   — Визуал > текст: графики, иконки, изображения.
   — 20 минут на 10 слайдов (Guy Kawasaki rule).
